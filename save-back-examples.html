Example files for trying out Save Back to File from DOM https://addons.mozilla.org/en-US/firefox/addon/save-back-to-file-from-dom/:

  • Traffic Light save-back-ex-001-traffic-light.svg click lights in traffic light
  • Pool Table save-back-ex-002-pool-table.svg drag balls on pool table

How to use: Store (save) these files on your local file system (disk). Open them from there. Makes changes (by using your mouse to click or to drag). Save back to file by using Save Back to File from DOM. Open again to see the changes have been saved.

Idea and first implementation by Leo Baschy and Hans Baschy.
